# Actix-Crud
Basic Actix + Diesel + Postgres REST API

# Setup
- Install and setup PostgreSQL
- Set `DATABASE_URL` environment variable or specify in .env file

```
git clone https://github.com/aathif394/actix-crud/
cd actix-crud
cargo run --release
```
